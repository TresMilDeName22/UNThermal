ThermalSystemIoT(broker_address='192.168.1.100', port=1883, plant_number='PC-T-00', user='user', password='user', client_id='', clean_session=True)

Class to manage a thermal system using IoT and MQTT protocol.

Parameters:

  • broker_address (str,required) – The MQTT broker’s address attending the thermal system experiment. For instance, “192.168.1.100”.

  • port (int, required) – The port for MQTT connection. Default is 1883.

  • plant_number (str, required) – The plant id assigned to the user’s specific thermal system. For instante “PC-T-11”.

  • user (str, optional) – The username for MQTT authentication in a remote experimental setup.

  • password (str, optional) – The password for MQTT authentication in a remote experimental setup.

  • Attributes

  • -----------

clientmqtt.Client

The MQTT client used for connection and communication.

codesdict

A dictionary containing codes for various MQTT topics related to the thermal system.

User methods:

transfer_function()

Returns the thermal system’s transfer function using data from the last PRBS (pseudo random binary signal) experiment.

Example:

First, ensure that you have already imported the unthermal package as follows:

>>> import unthermal as ter

Then, create an IoT Thermal System object, as follows:

>>> my_system = ter.ThermalSystemIoT(plant_number="PC-T-01", broker_address="192.168.1.100")

You can use the transfer_function method to retrieve the system’s transfer function based on the most recent PRBS identification experiment performed on the thermal system.

>>> G = my_system.transfer_function()
>>> print(G)
>>> <TransferFunction>: sys[5]
>>> Inputs (1): ['u[0]']
>>> Outputs (1): ['y[0]']
>>>    1.182
>>> -----------
>>>  36.98 s + 1

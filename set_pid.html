set_pid(system, kp=10, ki=2, kd=0, N=5, beta=0)

Adjusts the controller parameters in a two-degree-of-freedom (2-DOF) PID controller.

This function sets the five parameters \(k_p\), \(k_i\), \(k_d\), \(N\), and \(\beta\) of a 2-DOF PID controller, as depicted in the figure below:

Parameters:

  • system (ThermalSystemIoT) – The IoT thermal system on which the experiment will be performed.

  • kp (float) – Proportional gain of the PID controller. Default is 10.

  • ki (float) – Integral gain of the PID controller. Default is 2.

  • kd (float) – Derivative gain of the PID controller. Default is 0.

  • N (float) – Filter coefficient. Default is 5.

  • beta (float, optional) – Setpoint weight of the PID controller. Default is 0.

Returns:

Returns True once the PID parameters are successfully updated.

Return type:

bool

Notes

  • When \(\beta=1\), the resulting PID corresponds to the standard one-degree-of-freedom (1-DOF) PID described in most textbooks.

  • This implementation uses the algorithm formulated in the book Feedback Systems: An Introduction for Scientists and Engineers, Second Edition by Åström and Murray. The algorithm uses a sampling time of 0.8 seconds, suitable for the thermal system.

Example

First, ensure that you have already imported the unthermal package and defined the thermal system as follows:

>>> import unthermal as ter
>>> my_system = ter.ThermalSystemIoT(plant_number="XXXX", broker_address="192.168.1.100")

Then, the PID controller is programmed as follows:

>>> ter.set_pid(my_system, kp=5, ki=2, kd=0.1, N=5, beta=1)
